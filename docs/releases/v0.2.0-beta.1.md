# Mythos Writer v0.2.0-beta.1 — Windows Owner Preview

> *A writing app, with an extra brain, to keep everything in mind so you don't have to.*

Your manuscript lives in one tool. Your characters, locations, and factions live in another. Every time you introduce a new faction in chapter 12, you update it in two places. **Mythos Writer fixes that.** One local Windows app: write your scenes in the Story Vault, let the Brainstorm AI fill your Notes Vault with the characters and places it finds in your conversations, map your plot on the new Scene Crafter Kanban board, and follow the connections between your notes in the Vault Graph. Everything stays on your machine — no account, no subscription required to start.

This is an **unsigned Windows beta built for owner testing.** Install it, put it through its paces, and file bugs. That feedback shapes what ships next.

---

## What's new in v0.2.0-beta.1

### Scene Crafter Kanban board
Drag scene cards across columns to plan your story visually. Each card links back to the scene file in your vault; the board itself is stored as a Markdown file you can edit in Obsidian or any text editor.

### Vault Graph
A live map of every `[[WikiLink]]` connection across your Notes Vault. Follow the thread from one character to every location, faction, and event they appear in. Includes keyboard-accessible navigation.

### Getting Started panel
A dismissible guided checklist in the right sidebar walks you through writing your first scene, adding a character, trying Brainstorm, and exploring the Notes Vault. Disappears once you're ready to fly solo.

### Brainstorm Panel: drag-reorder + Open in Writing Panel
Sort ideas in Custom mode by dragging them into the order you want. Send any Brainstorm idea directly to the linked scene via **Open in Writing Panel** — one click, no copy-paste.

### Custom template management
Save your own vault structure as a reusable template, then rename, duplicate, delete, or import/export it alongside the four bundled templates (Novel, Short Story, World-building Bible, Series Bible).

### Archive Agent v1 — Continuity Panel
The Archive Agent now surfaces inconsistencies it finds between your manuscript and your Notes Vault. Each finding shows up in the Continuity panel as a checkbox item you can review and resolve inline.

## Core features in this build

| Feature | What it does |
|---------|--------------|
| **Story Vault** | Stories → Chapters → Scenes; each scene is a Markdown file you own |
| **Notes Vault** | Free-form Markdown folder for characters, locations, factions, and worldbuilding |
| **Obsidian-compatible** | Open any Mythos Writer vault in Obsidian, or import an existing Obsidian vault |
| **Rich scene editor** | TipTap block engine with WikiLinks (`[[Character Name]]`), draft states, and live word count |
| **Writing modes** | Normal, Focus (distraction-free), Edit (inline AI suggestions); toggle with `Ctrl+Shift+N/F/E` |
| **Brainstorm AI** | Conversational story development; auto-extracts entities into your Notes Vault |
| **Writing Assistant** | Proactive suggestions sidebar on a configurable heartbeat; never edits your text |
| **Entity browser** | Characters, locations, factions, items, events, concepts — wiki-linkable from any scene |
| **EPUB + DOCX export** | File → Export… |
| **Snapshot history** | Automatic per-scene version snapshots; right-click the editor to restore any previous state |
| **Project templates** | Four bundled structures (Novel, Short Story, World-building Bible, Series Bible) + your own |
| **Voice input** | Local-first STT; optional cloud STT/TTS via any OpenAI-compatible voice endpoint |
| **AI providers** | Anthropic (default), OpenAI, Ollama, LM Studio, or any OpenAI-compatible endpoint |
| **Vault status badge** | Shows whether your vault lives locally or inside a synced folder (Dropbox, Google Drive, etc.) |

## Known limitations (beta)

**Unsigned installer.** Windows SmartScreen will warn you before install. Click **More info → Run anyway**. This is expected for owner-testing builds; code signing is on the roadmap ([SKY-1670](/SKY/issues/SKY-1670)).

**AI features require your own API key.** Bring an [Anthropic key](https://console.anthropic.com/) (default) or configure a different provider under Settings → AI Provider. Keys are stored locally and never leave your machine.

**No packaged macOS release.** macOS and Linux stay green in CI and are build-tested every PR; packaged releases follow once Windows is stable in users' hands.

**No auto-updater in this build.** Future betas will ship via the built-in updater. For now, download new releases from this page manually.

**Voice input is off by default.** Enable it under Settings → Voice. Local STT works on device; cloud STT/TTS requires an OpenAI-compatible voice provider.

**Archive continuity checks are additive only in v1.** The Continuity panel surfaces findings but does not yet propose automatic corrections. Resolve each finding manually via the checkbox list.

**This is a beta.** Cosmetic issues, edge-case crashes, and placeholder copy exist and are being actively resolved. Expect rough edges.

## Install

**→ Download `Mythos.Writer-0.2.0-beta.1.exe` from the assets below.**

1. Run the `.exe`.
2. If Windows SmartScreen appears, click **More info → Run anyway**.
3. Launch **Mythos Writer** and complete the onboarding wizard. The Getting Started checklist will guide you from there.

**Found a bug or have feedback?** [Open a GitHub issue](https://github.com/SkyyPlayz/Mythos-Writer/issues) — describe what you did, what you expected, and what happened. Screenshots are very helpful.
