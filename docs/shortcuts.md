## Intentionally Absent Shortcuts

The following common shortcuts are **not** wired in Mythos Writer by design:

| Chord | Reason |
|-------|--------|
| `Ctrl / Cmd + S` | Autosave is always on — explicit save is unnecessary. |
| `Ctrl / Cmd + N` | New note is toolbar-only to prevent accidental creation. |
| `Ctrl / Cmd + F` | Global find relies on focusing the SearchBar (no modal). |
